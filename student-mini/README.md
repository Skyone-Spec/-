# 学院学生综合服务与党团管理平台 - 学生端小程序

## 项目说明

基于微信小程序开发的学生端应用，采用原生开发模式，配合 Vant Weapp 组件库实现 UI。

## 技术栈

- **框架**: 微信小程序原生开发
- **UI组件库**: Vant Weapp
- **开发工具**: 微信开发者工具

## 项目结构

```
student-mini/
```

## 功能模块

### UC01 登录与身份绑定
- 微信登录 + 学号绑定
- 防抖处理 + 失败锁定机制

### UC02 智能检索与知识库
- 关键词搜索
- 分类筛选（政策/通知/FAQ/模板）

### UC03 问答工单
- 提交问题
- 查看回复

### UC04 证明/办事申请
- 申请表单
- 附件上传（≤20MB）
- 进度追踪

### UC07 党团流程追踪
- 入党/入团进度可视化
- 待办任务清单

### UC08 通知中心
- 三类消息（待办/反馈/个性化）
- 未读标记

### UC09 学业分析与预警
- 成绩单上传
- 学情图表展示

### 个人中心
- 个人信息
- 我的申请/工单/党团
- 意见反馈

## 运行项目

1. 克隆代码到本地
2. 使用微信开发者工具打开项目
3. 填写 `project.config.json` 中的 AppID
4. 编译运行

## 注意事项

- 确保已安装 Vant Weapp 组件库
- 需要配置合法域名（开发阶段可关闭域名校验）
- 分包加载需在 app.json 中正确配置
